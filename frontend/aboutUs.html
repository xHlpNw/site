Учебный проект

AutoMarket — каталог подержанных автомобилей

Веб-приложение, созданное с использованием HTML, CSS и JavaScript.
Разработано для демонстрации навыков адаптивной вёрстки и интерактивного интерфейса.

Цели проекта

Проект создан для практического применения базовых технологий веб-разработки — HTML, CSS и JavaScript. Основная цель — создание адаптивного и современного интерфейса каталога автомобилей без использования фреймворков.

Освоение классического подхода к фронтенд-разработке

Технологии

Проект построен на базе стандартного стека фронтенд-разработки с применением новейших возможностей языков: структура выполнена на HTML5, оформление — на CSS3, логика и интерактивность реализованы с помощью JavaScript (ES6+). Для организации кода использованы семантические теги, кастомные CSS-переменные и модульный JS.

HTML5 CSS3 JavaScript (ES6+)

Особенности проекта

Ключевые технические решения, реализованные с использованием базовых технологий фронтенда.

Безопасный код

Используются встроенные возможности HTML5 для валидации форм и защиты данных.

Удобный интерфейс

Современная адаптивная вёрстка с использованием CSS Flexbox и Grid.

Быстрая загрузка

Минимум внешних зависимостей и оптимизация кода обеспечивают быструю загрузку страниц.

Принцип работы

Демонстрация основного пользовательского сценария в приложении

01

Поиск

Пользователь выбирает параметры для поиска нужного автомобиля.

02

Выбор

Просмотр детальной информации и фотографий выбранного авто.

03

Покупка

Получение контактов и оформление сделки напрямую.
